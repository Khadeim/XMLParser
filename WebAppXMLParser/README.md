XML Parser - Khadeim Rahman

# Project Description

This is a Python-based XML parser designed to extract structured data from XML files. For demonstration purposes, it uses an XML file representing Arsenal Football Club methods and attributes. The parser converts XML content into a JSON format, making it easier to process and use in other applications.

# Why I Built It

To practice parsing XML files using Python and BeautifulSoup.
To demonstrate how data can be extracted, structured, and transformed for analysis or integration with other systems.
To create a simple, self-contained project that showcases problem-solving and scripting skills.

# Skills Demonstrated

Python programming (file handling, loops, dictionaries, functions).
Data parsing with BeautifulSoup for XML content.
JSON formatting for clean, structured output.
Debugging and testing with different XML input examples.

Import the 'Beautiful Soup' library shown in the documentation below:
https://pypi.org/project/beautifulsoup4/

pip install cmd line command: pip install beautifulsoup4
pip install lxml beautifulsoup4

Run using: 'python WebAppParser.py' in terminal
This will convert the hard coded XML file which is currently 'Arsenal.xml' for demonstration
